## 🎨 Professional Identity

This portfolio is a showcase of my technical skills, creative vision, and dedication to building high-quality digital products. It features a unique **3D gaming-style aesthetic** with smooth animations and a premium dark theme.

### 🔥 Key Features
- 🌌 **3D Interactive Background:** Powered by Three.js and React Three Fiber.
- ⚡ **Neon Green Aesthetic:** A professional, state-of-the-art dark mode design.
- 📱 **Fully Responsive:** Optimized for Mobile (Android/iOS) and Desktop.
- ⏳ **Gaming Loading Screen:** Custom terminal-style boot sequence (1% - 100%).
- 📥 **Interactive CV Download:** Animated fetching process for resume payload.
- 🚀 **Lightning Fast:** Built with Vite and React 19 for maximum performance.
